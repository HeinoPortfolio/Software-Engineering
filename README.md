# Software-Engineering

##  Description
> Software engineering principles and concepts. The software life cycle, structured specifications, design tools and techniques, software reliability, and verifying program correctness.

-------

## Competencies
Competencies and objectives completed:

- $`\textcolor{blue}{\text{Elicit vague requirements from a client partner.}}`$
- $`\textcolor{blue}{\text{Analyze the requirements and make clarifications in the requirements.}}`$
- $`\textcolor{blue}{\text{Select the most appropriate software development methodology with respect to the size of the project and needs of the client partner.}}`$
- $`\textcolor{blue}{\text{Apply object-oriented design principles, design patterns, and the Model-View-Controller architecture.}}`$
- $`\textcolor{blue}{\text{Design the project in the form of documents, UML diagrams, sketches, prototypes, drawings.}}`$
- $`\textcolor{blue}{\text{Implement the project under the change requests from the client partner.}}`$
- $`\textcolor{blue}{\text{Verify and validate the project whether it solves the client partnerâ€™s problem.}}`$
- $`\textcolor{blue}{\text{Make a sustainable maintenance plan.}}`$
-------
